## Key decisions
- PDF via `window.parent.print()` in an `height=0` iframe — browser-native, no server-side PDF lib needed
- `_db_is_ready()` checks row count not just file: Streamlit Cloud creates an empty DB file before init completes
- Auto-reinit in `except ValueError` block catches container-restart DB wipes without user action
- Start date locked to `2021-01-01` on both cloud and local — ensures identical optimization results
- VN30 list updated per official Q1/2026 rebalance announcement — keeps data current for thesis defense
